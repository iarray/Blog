在定义变量前加typedef, 变量名将代表变量类型, 即相当于给变量类型起一个别名
```c
//定义一个变量byte
unsigned char byte;
//将unsigned char用byte来表示;
typedef unsigned char byte;

```

通常情况下
```c
//整形占4字节
printf("sizeif(int)=%d\n", sizeof(int));
//指针在,32位系统占4字节, 64位操作系统占8字节
printf("sizeif(int*)=%d\n", sizeof(int*));
//byte占1字节
printf("sizeif(byte)=%d\n", sizeof(byte));

```
