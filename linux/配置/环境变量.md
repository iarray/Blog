### 查看环境变量
```shell
env
```
或者 
```shell
printenv
```

### 添加环境变量(临时起效)
```shell
export KEY=VALUE
```
下面以nodejs为例
```shell
export NODE_HOME=/opt/node

export PATH=$PATH:$NODE_HOME/bin 

export NODE_PATH=$NODE_HOME/lib/node_modules
```

### 添加环境变量（永久）
```shell
vim /etc/profile
```
在末尾添加
```
export NODE_HOME=/opt/node

export PATH=$PATH:$NODE_HOME/bin 

export NODE_PATH=$NODE_HOME/lib/node_modules

```
保存后， 执行
